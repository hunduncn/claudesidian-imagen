# @claudesidian/imagen

Local web tool to turn Claudesidian vault .md articles into 小红书/公众号 covers and illustrations.

## Quick Start

```bash
cd /path/to/your-claudesidian-vault
bunx @claudesidian/imagen
```

Opens http://127.0.0.1:5173 in your browser.

See `01_Projects/claudesidian-imagen/2026-04-18-设计文档.md` in any Claudesidian vault for full design.
